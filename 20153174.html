  3. Describe what do the following tags mean(4 point)
    • pre: Preformatted 이 태그는 문장의 위치와 간격을 작성한 그대로 화면에 뿌려주는 효과를 나타내 줍니다. 단, 태그는 본래의 태그로 그대로 효과를 나타냅니다

  4. Describe diffrence between using strong, em tag and b, i tag (5 point). Answer: em이나 strong으로 되어 있으면 웹브라우저 뿐만 아니라 다른 음성 읽기 도구에서도 내용을 강조해서 읽어준다. 일반적으로 사용자에게 보여줄 때 em이탤릭체, strong은 굵은 글씨로 보여주며 의미를 강조한다고 되어 있다. i태그와 b태그는 스타일로 단지 의미보단 표면적으로 보여질 때만 사용 되는 것이다. b나 i로 되어 있으면 그저 웹브라우저에서만 문자열을 굵게 표시하거나 효과를 주는 것이다.
  5. In the img tag, describe what alt tag means and how it differs from title attribute(5 point). Answer: Alt태그 또는 Alt속성은 이미지의 “대체문구”(alternative text)입니다. 보통 Alt태그는 이미지를 설명하는 목적으로 만들어집니다. 웹사이트라면 Alt태그는 사람들에게 보여지지 않습니다. 웹사이트에서 Alt태그가 사용되는 경우가 한가지 있는데, 시각장애인이 음성기반의 소프트웨어로 웹서핑을 할 때 쓰입니다. 반면에 다른 대부분의 사람들을 위해서는 검색엔진에 사용됩니다. *title(tooltip attribute)은 이미지에 대한 툴팁 설명을 제공하는 속성 *alt(alternate attribute)은 이미지를 대체하는 문구를 제공하는 속성
  6. In table, describe difference between th and td (5 point). Answer: *th: table head의 약자로 표의 제목을 쓰는 역할(기본값은 굵은 글씨체에 중앙 정렬) *td: table data의 약자로 셀을 만드는 역할이다. (기본값은 보통 글씨체에 왼쪽 정렬이다.) 즉, td는 테이블의 데이터를 의미하고 th는 해당 테이블의 항목에 대한 설명을 의미한다.
